Takeaways
==============================================================
  Strong points:
   • AUC / accuracy / F1 / params / size all match RESULTS.md
     and CLAUDE.md exactly — no silent drift.
   • Violence-dataset clips detected best (TPR 96%) — the signal
     the model was trained on comes through strongly.
   • Gunshot (89%), glass breaking (80%), explosion (74%) are all
     robust even though they are absent from RESULTS.md's table.
   • Synthetic ambient / quiet / silence = 0% FPR; applause,
     cheering, percussive hits (hns) stay under 14% FPR.

  Weak points:
   • Laughter (82.5%) is actually a LARGER false-alarm source
     than speech (71.7%) — docs call out speech but not laughter.
   • Shouts are the weakest threat class (TPR 64.7%) — blend with
     angry conversation.
   • Non-violent physical interaction (viol_violence, safe side)
     triggers 60.7% FPR — body/impact sounds overlap with threats.
   • The ROC-AUC looks OK (0.80) but max balanced-accuracy is
     only 0.733 — there is NO clean operating point, every
     threshold trades a lot of TPR for a little FPR.

  Quirks worth knowing:
   • The ~12ms CPU latency claim does NOT reproduce on this
     machine — measured value is printed above. Likely a
     per-hardware / per-thread-count figure; any deployment
     claim should be re-measured on target hardware.
   • Threat confidence is muted: unsafe samples have mean P=0.63
     and p10=0.38. 10% of true threats sit below the default
     threshold → built-in ~19% miss rate at thr=0.5.
   • Unsafe predictions are almost never sharp (1% > 0.9). The
     model 'hedges' — useful for threshold tuning, bad for a
     one-shot argmax decision.
   • To push FPR below 10%, threshold ≥0.70 is required, which
     slashes TPR to 37.5%. The 'low_fpr' threshold strategy in
     finetune.py exists precisely because of this curve shape.
   • 'viol_violence' appears in BOTH classes by design: the
     violence dataset bundles violent + non-violent interactions
     (see safecommute/pipeline/prepare_violence_data.py:6-7).
   • RESULTS.md per-source safe table only lists 4 sources
     (speech, crowd, laughter, metro) and omits 12 others,
     including the very clean ones that would strengthen its
     deployment story.
