# SafeCommute AI — Web

Bauhaus-inspired landing page for SafeCommute AI. Next.js 14 (App Router) + TypeScript + Tailwind + motion/react.

## Preview

```bash
cd web
npm install
npm run dev
# http://localhost:3000
```

## Build (static export)

```bash
npm run build
# output in web/out/ — deploy anywhere static
```

## Deploy

- **Vercel**: `vercel` from `web/`.
- **Netlify**: drag `web/out/` into the Netlify dashboard.
- **GitHub Pages**: push `web/out/` to a `gh-pages` branch.

## Editing copy

All marketing copy lives inline in `components/*.tsx` — open the component, edit the strings, save.

Pricing numbers: `components/pricing.tsx` → `TIERS`.
Per-source accuracy: `components/honesty-block.tsx` → `THREATS` / `HARD`.
Team: `components/team-section.tsx` → `MEMBERS`.
Mailto/GitHub links: `lib/utils.ts`.

## Figures

Figures are copied from `../docs/figures/`. When pitch figures are regenerated by `scripts/generate_pitch_figures.py`, rerun:

```bash
cp ../docs/figures/*.png public/figures/
```
